# C-qt-GroupChat
包含客户端和服务器，客户端分为登陆界面和聊天界面，能够实现局域网多人聊天，显示时间，历史记录加载以及注册账号和记住密码的功能Includes client and server. The client is divided into login interface and chat interface, capable of LAN multi-user chat, displaying time, loading chat history, as well as registering accounts and remembering passwords.
